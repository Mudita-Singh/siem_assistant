# 🛡️ SIEM Assistant - Cybersecurity Dashboard

Welcome to the **SIEM Assistant**! This is a modern, interactive Security Information and Event Management (SIEM) dashboard designed to help security analysts monitor, analyze, and respond to threats in real-time.

> [!IMPORTANT]
> **Status**: 🚧 Currently under active development.  
> **Affiliation**: This project is being built as part of my **College Minor Project**.

---

## 📊 Features

- **Real-time Dashboard**: Monitor system health, threat levels, and active alerts at a glance.
- **AI-Powered Assistant**: An integrated chat assistant to help with security queries and log analysis.
- **Alert Management**: A dedicated panel for viewing and mitigating security incidents.
- **Detailed Logs**: Comprehensive logging system with filtering and search capabilities.
- **Reports & Analytics**: Generate visual reports on security trends and system performance.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Charts**: Chart.js / React-ChartJS-2
- **Icons**: Lucide React
- **Build Tool**: Vite

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Mudita-Singh/siem_assistant.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd siem_assistant
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Start the development server**:
    ```bash
    npm run dev
    ```
